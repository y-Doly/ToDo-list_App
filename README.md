# To-Do List Web Application with Affirmations

A simple and beginner-friendly **To-Do List web application** built using **HTML, CSS, and JavaScript**.  
This app allows users to add and delete tasks and also displays a **motivational affirmation** that changes every time the page is refreshed.

---

## Features

1. Add new tasks
2. View tasks instantly
3. Delete tasks
4. Daily affirmations for motivation
5. Affirmation changes on every page refresh
6. Clean and responsive UI
7. No external libraries or frameworks

---

## Technologies Used

1. **HTML5** – Structure of the application
2. **CSS** – Styling and layout
3. **JavaScript** – Logic and DOM manipulation
4. **LocalStorage** – To prevent repeating affirmations on refresh
